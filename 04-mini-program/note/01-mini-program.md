## Mini Program

#### what is mini-program

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


/**
 * mini-program                 // CX
 * 
 * 
 * < 2M
 */

1) 微信小程序，是一种不需要下载安装即可使用的应用

2) 实现了应用 '触手可及' 的梦想，用于 '扫一扫' 或 '搜一下' 即可打开

3) 体现了 '用完即走' 的理念，用户不关心是否安装太多应用的问题
```
